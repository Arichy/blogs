# 3. Setup SigNoz

最推荐的方式是使用 Docker Compose 来安装 SigNoz, 参考官方文档:
https://signoz.io/docs/install/docker/#install-signoz-using-docker-compose

```bash
# 1. clone repo
git clone -b main https://github.com/SigNoz/signoz.git && cd signoz/deploy/

# 2. 运行 docker compose
cd docker
docker compose up -d --remove-orphans
```

根据官网的指示, 当执行 `docker ps` 看到以下 4 个 container, 就说明成功了:

接下来访问 `http://localhost:8080`, 就可以看到 SigNoz 的 UI 了. 初次访问可能会让你注册一个账号登录, 随便填一个账号密码都行, 因为这个是完全存在本地的.

注册完之后应该能进入 home 页 (默认应该是深色主题, 我在设置里手动改成了浅色, 这个不重要)
